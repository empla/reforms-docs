Reforms.js

Vue 3 and Bootstrap 5 forms and cards generator

Quick Start →

🗂 Forms and Cards

You can generate forms and cards (it's a readonly form) with 15+ inputs and outputs

👮🏽‍♀️Validation

Supports client side and server side validation

🌏 Internationalization

You can internationalize validation messages

🌱 Fields grouping and sorting

Fields can be grouped and sorted

🔌 Plugin system

Can create custom input/output type and validators

✅ Written for Vue 3 and Bootstrap 5

This library depends on Vue 3 and Bootstrap 5
